# House-Price-Prediction-System
This project predicts house prices using Machine Learning based on
- Area (sq ft)
- Number of bedrooms
- Age of the house

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## ML Workflow
- Data preprocessing
- Feature scaling
- Train-test split
- Linear Regression
- Model evaluation (RÂ², RMSE)
- Web app deployment using Streamlit

## How to Run
pip install -r requirements.txt  
streamlit run app.py
